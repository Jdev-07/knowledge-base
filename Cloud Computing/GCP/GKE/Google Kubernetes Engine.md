
# Overview

**What is Kubernetes?** 

Kubernetes is an orchestration framework for software containers. Containers are the way to package and run code in way more efficient than virtual machines. Kubernetes provides tools you need to run containerized applications in productions and at scale. 

**What is GKE?**

Google Kubernetes Engine (GKE) is a managed service for Kubernetes.

GKE runs containerized applications in a cloud environment. Also, allows to package the code and all its dependencies. 

**What cloud computing is?**

Cloud computing is a way of using IT computing resources that customers can get on demand and access anywhere in the world. These resources are elastic (flexible), then customer pay only for what they use or reserve as they go.

Note: 

- The resources hierarchy directly relates to how policies are managed and applied to google
- Billing is established at project level
- A billing account cab be linked to zero or more projects.
- Billing accounts are charged automatically and invoiced every month, or at every threshold limit
